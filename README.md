A# AFFCM

**AFFCM** (AI Full File Control Manager) is a Java-based desktop application that allows users to leverage custom AI models to organize, sort, and manage files intelligently. Built with **LLaMA.cpp**, AFFCM enables personalized AI models chosen by the user, providing powerful recommendations and automation for file management.

---

## Features

- **Custom AI Models**: Use your own AI models to guide file sorting and organization.  
- **Folder Recommendation**: Automatically suggest the best existing folders for files based on AI predictions.  
- **File Organization Automation**: Quickly organize your files without manual intervention.  
- **Java-Based**: Fully built in Java, leveraging modern features and libraries.  
- **Integration with LLaMA.cpp**: Efficient AI inference with popular open-source LLaMA models.
- **File Structure**: Includes model-info.java file for perfect structured application.
- **Multithreading Support**
